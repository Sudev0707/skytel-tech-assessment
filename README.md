
# Tech Blogs App

## Features
- SEO optimized using react-helmet-async
- Structured Data (JSON-LD)
- 10 blogs fetched from API
- Search across title, description and content
- Category filter
- Fully responsive UI

## Technical Decisions
- Used HelmetProvider to inject meta tags dynamically
- Implemented Article JSON-LD schema for SEO rich results
- Used WebSite schema for homepage
- Used TypeScript for strict type safety
- Used Vite for faster development

## Running Lighthouse
Open app → Ctrl + Shift + I → Lighthouse → Generate report
